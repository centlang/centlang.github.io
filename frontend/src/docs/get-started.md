# Getting Started

## Cent Programming Language

Cent is a systems programming language designed to be simple and efficient.

### Performant

Thanks to LLVM, programs written in Cent are as fast as those written in C and
C++.

### Modern

Cent aims to include most modern programming features without sacrificing
performance.

### Fast compilation

Cent is very straightforward, which makes for faster compilation times.
